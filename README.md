## Описание проекта

Это графическое приложение (GUI), которое помогает выбрать случайное занятие из предопределенных категорий (Учёба, Спорт, Работа). Приложение позволяет добавлять собственные задачи и сохраняет историю сгенерированных вариантов.

**Основные возможности:**
*   **Генерация:** Выбор случайной задачи из общего списка или конкретной категории.
*   **Фильтрация:** Возможность ограничить выбор одной из категорий (Учёба/Спорт/Работа).
*   **История:** Отображение всех ранее сгенерированных задач в текущей сессии.
*   **Расширение:** Добавление пользовательских задач в список.
*   **Сохранение:** История задач сохраняется в файл `data/history.json`.

## Инструкция по запуску

### 1. Установка зависимостей

Для работы приложения требуются стандартные библиотеки Python (`tkinter`, `random`, `json`, `os`). Библиотека `requests` **не требуется**.
Никаких сторонних зависимостей устанавливать не нужно.
